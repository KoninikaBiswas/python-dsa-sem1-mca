# Python DSA Programs - MCA Sem 1

Python programs covering fundamental Data Structures & Algorithms. Built during MCA 1st semester at Siliguri Institute of Technology.

**Note**: Practiced 30+ Python programs during sem-1. This repo contains 12 key DSA & algorithm implementations.

### Programs Included
**Searching & Sorting**
- Binary search.py - O(log n) search algorithm
- Linear search.py - Linear search implementation  
- Bubble sort.py - Bubble sort algorithm
- Merge sort.py - Divide & conquer O(n log n) sort
- Selection sort.py - Selection sort algorithm

**Recursion & Number Theory**
- factorial using recursion.py - Recursive factorial
- fibonacci using recursion.py - Recursive fibonacci
- gcd.py - GCD calculation
- LCM of two numbers.py - LCM calculation
- Armstrong using function.py - Armstrong number check

### Tech Stack
Python 3.x

### How to Run
`python "Merge sort.py"`
